# Como executar o projeto Notas

Este projeto foi desenvolvido utilizando **Flutter** e pode ser executado através do **Visual Studio Code (VS Code)**.

## 1. Instalar o Visual Studio

Antes de instalar o Flutter, é necessário ter o **Visual Studio Community** instalado no computador.

1. Acesse o site oficial da Microsoft.
2. Baixe o **Visual Studio Community** (versão gratuita).
3. Realize a instalação normalmente no seu computador.

## 2. Instalar o Flutter no VS Code

1. Abra o **Visual Studio Code**.
2. Pressione **Ctrl + Shift + P** para abrir a paleta de comandos.
3. Digite **Flutter** na barra de pesquisa.
4. Clique na opção **Flutter: New Project**.
5. O VS Code solicitará que você localize o **Flutter SDK** no seu computador.

### Baixando o SDK

1. Clique em **Download SDK**.
2. Escolha a pasta onde deseja instalar o Flutter.
3. Clique em **Clonar Flutter**.
4. Aguarde o download ser concluído (isso pode levar alguns minutos).

Após a instalação terminar, **reinicie o computador** para que todas as configurações sejam aplicadas corretamente.

## 3. Baixar o projeto

1. Baixe o arquivo **notas.zip**.
2. Extraia o conteúdo do arquivo.
3. Entre na pasta extraída.
4. Retire a pasta **notas** de dentro do arquivo compactado.
5. Coloque a pasta **notas** em um local de fácil acesso no computador.

⚠️ Importante:
Não execute o projeto diretamente dentro do **notas.zip**, pois isso pode causar erros.

## 4. Abrir o projeto no VS Code

1. Abra o **Visual Studio Code**.
2. Clique em **File → Open Folder**.
3. Localize a pasta **notas**.
4. Selecione a pasta e clique em **Selecionar Pasta**.

O VS Code carregará todos os arquivos do projeto.

## 5. Executar o projeto

1. No VS Code, clique em **Terminal → New Terminal**.
2. No terminal que abrir, digite o comando:

*flutter run*

3. O sistema pedirá para escolher um **navegador** para executar o projeto.
4. Selecione o navegador desejado.
5. Aguarde alguns segundos até que o sistema seja aberto.

## Pronto!

O projeto **Notas** será executado e estará pronto para uso.
